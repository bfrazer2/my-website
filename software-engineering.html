On this page you'll find some examples of projects I've built, along with a brief explanation.

Steam Auto Seller

This project is a webscraper which sells items in a user's steam inventory automatically.

It installs a browser extension which scrapes for the lowest possible current market price of an item as well as the highest possible instant-sell price, and generates buttons for those options.

The scraper I've built then accepts the user's login credentials and 2 factor authentification code, and allows the user to custom select filter options. The scraper will then sell all items matching the selected filter conditions, preferencing the instant-sell option.

Most Recent 20

This project started as a webscraper but now uses the spotify developer API as it became apparent it was a much more efficient solution.

The program was built for my partner, and takes the most recently added 20 songs in their main spotify playlist and adds them to a playlist consisting of only those songs.

It will check for duplicates, and remove any existing songs in the most recent 20 playlist which no longer belong.
Blackjack

A project standard, this program allows the user to play a game of blackjack against an AI dealer.

It does allow for local multiplayer, and the game is played in the console.

The code includes text-decoration to ensure that the console is readable & pleasant.
